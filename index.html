Projects

What The Happening?

What the Happening is a web app that allows users to input and view events, including a time, description, and map location into the site's database. Events are displayed in chronological order on the left sidebar with location markers showing the user-inputted location. The website was built with Express.js, Node.js, AngularJS, Bootstrap, and the Google Maps API.

Dartboard

This website aims to generate the ideal trip given the location you will be staying. The site caluates the best day plans for the entire trip, accounting for attraction ratings and travel time. Finally, planning to travel to a foreign place is simple and stress free. Dartboard utilizes Flask, Bootstrap, jQuery, Bing Maps API and Yelp API.
